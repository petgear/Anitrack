# Anitrack

Anitrack — сервис для поиска аниме и ведения списков
(просмотрено / запланировано / заброшено).

Проект создаётся как учебно-практический, с фокусом на:

- архитектуру React-приложения
- работу с TanStack Query и MobX
- аккуратный UI и темизацию

---

## О проекте

Цель проекта - показать:

- Работа с архитектурой React (FSD)
- Использование Tailwind CSS
- Верстка и UI с использованием MUI
- Понятная и масштабируемая структура проекта
- Работа с глобальным состоянием (MobX)
- Работа с REST API через TanStack Query

## Установка и запуск

1. Клонировать репозиторий:

```bash
git clone https://github.com/petgear/Anitrack
```

2. Перейти в директорию проекта:

```bash
cd Anitrack
```

3. Установить зависимости:

```bash
npm install
```

4. Запустить локальный сервер:

```bash
npm run dev
```
